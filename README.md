# 🌑 Planalto Sombrio

## 🎮 Sobre o Jogo

**Planalto Sombrio** é um jogo de defesa em ondas desenvolvido em Unity 2022.x usando C#.  
O jogador deve proteger o “planalto” — uma estrutura central com 6 vidas — contra inimigos que avançam em três ondas progressivamente mais difíceis.

### ✔️ Estrutura das Ondas
- **Onda 1:** 2 bosses  
- **Onda 2:** 3 bosses  
- **Onda 3:** 4 bosses  
- Ao concluir a terceira onda, surge uma **tela de vitória**.

### ✔️ Defensores
O jogador começa com um defensor básico que atira garrafas automaticamente.  
Durante o jogo, é possível desbloquear mais dois personagens:

- **Influencer** – atira tigrinhos estilizados  
- **Político** – lança cuecas com dinheiro  

Todos atacam automaticamente o inimigo mais próximo.

### ✔️ Poder Manual: Raio
O jogador pode clicar em um inimigo para invocar um **raio**:
- derrota inimigos fracos instantaneamente  
- causa metade do dano nos mais resistentes  
- possui **cooldown reduzido** na versão final  

### ✔️ HUD (Interface)
A interface mostra:
- contador da onda atual  
- contador de quantos inimigos já foram derrotados  
- cooldown do raio  
- vidas do planalto

## 📦 Como Executar (Windows 10/11)

1. Baixe o arquivo:
PlanaltoSombrio.zip

2. **Extraia tudo** para uma pasta no seu computador.

Depois de extrair, a pasta deve conter pelo menos:
PlanaltoSombrio.exe
PlanaltoSombrio_Data/
UnityPlayer.dll

3. **Execute o jogo** abrindo:
PlanaltoSombrio.exe

4. Se o Windows mostrar aviso de segurança:

Mais informações → Executar mesmo assim

## 🧑‍💻 Como Abrir o Projeto na Unity (Opcional)

1. Abra o Unity Hub  
2. Clique em **Open**  
3. Selecione a pasta principal do projeto  
4. Abra a cena:
Assets/Scenes/MainScene.unity

5. Pressione **Play** para testar
